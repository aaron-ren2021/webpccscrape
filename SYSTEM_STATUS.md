## 📞 故障排除

### 來源全部失敗
1. 檢查網路連線
2. 確認 Playwright 已安裝：`playwright install chromium`
3. 檢查 .env 設定

### 標案數量異常少
1. 執行 `python check_sources.py` 檢查各來源
2. 查看 logs/cron.log 的 `source_failed` 訊息
3. 檢查 selector 是否失效（網站改版）

### 執行時間過長
1. 檢查是否等待 detail 頁 CAPTCHA（應該快速放棄）
2. 確認 stealth cooldown 設定合理（5-15 秒）
3. 查看 log 中的 batch_fetch_error 數量
