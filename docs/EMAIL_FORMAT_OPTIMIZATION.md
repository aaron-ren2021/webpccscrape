# 信件格式優化說明

## 優化日期
2026-04-07

## 最新更新（2026-04-07）
- **移除公告日期欄位**：從卡片中移除「📅 公告日期」，只保留「⏰ 截止投標」和「📌 開標時間」
- **理由**：公告日期是過去資訊，對投標決策影響較小，移除後減少視覺雜訊，提升閱讀效率
- **移除 AI 分析輸出**：暫時移除郵件中的 AI 摘要、分析理由和優先度徽章顯示
- **理由**：AI 分析功能尚未實現，避免顯示空白或無意義的內容

## 優化內容

### 1. 主旨優化
- **新增最緊急截止時間顯示**
  - 原格式：`[教育資訊標案] 2026-04-07 新增 5 筆`
  - 新格式：`[教育資訊標案] 2026-04-07 新增 5 筆｜最緊急截止 2026-04-10`
  - 自動從所有標案中找出最早的截止日期，直接在主旨中標示

### 2. 結構層面優化
- **信件 Header**
  - 明確標示「📧 教育資訊標案每日監控」
  - 顯示「系統自動通知，請勿直接回覆」說明
  
- **卡片式排版取代表格**
  - 每筆標案獨立卡片區塊（`.bid-card`）
  - 垂直排列清晰易讀，無需橫向滾動
  - Hover 效果提升互動體驗
  - 手機版響應式排版（@media query）

- **Footer 收尾說明**
  - 提醒「本郵件由標案監控系統自動寄出，請勿直接回覆」
  - 如有問題請聯繫系統管理員

### 3. 資訊層面優化
- **截止投標時間**：紅色字體 (`.deadline-red`) 突顯緊急性
- **預算金額**：加粗綠色 (`.amount-bold`) 便於業務判斷
- **押標金推導**：當押標金為 0、空字串或「無」時，自動顯示綠色「免繳」標示
- **軟體標籤**：有「軟體」標籤的標案使用特殊綠色底色 (`.tag.software`)

### 4. 視覺設計
- **Emoji 圖示**：增加可讀性（🏫 機關、💰 金額、⏰ 截止等）
- **連結按鈕**：藍色按鈕樣式「📄 查看詳情」取代純文字連結
- **顏色系統**：
  - 主色：藍色 (#3b82f6)
  - 金額：綠色 (#059669)
  - 截止：紅色 (#dc2626)
  - 背景：淺灰 (#f9fafb)

### 5. 響應式設計
```css
@media (max-width: 600px) {
  .card-row { flex-direction: column; }
  .card-label { min-width: auto; }
}
```
- 手機版自動調整排版
- 標籤元素垂直排列
- 減少橫向空間需求
